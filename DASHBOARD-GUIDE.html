üìö Step-by-Step Tutorials

Tutorial 1: Change Company Name

  1. Open Dashboard
  2. Go to General Settings
  3. Find "Company Name" field
  4. Enter new name (e.g., "MyTech Solutions")
  5. Click "Save" button
  6. Watch the name update on the website in real-time!
‚ú® Tip: The name appears in the logo area and page title. Changes sync instantly across all pages.

Tutorial 2: Change Brand Colors

  1. Open Dashboard
  2. Go to Design & Colors
  3. Click on color picker next to "Primary Color"
  4. Choose your desired color from color wheel
  5. Click "Apply"
  6. See website colors change instantly!
  7. Repeat for Secondary and Tertiary colors
üí° Pro Tip: Primary color is used most frequently on buttons and accents. Choose a color that represents your brand!

Tutorial 3: Edit Website Content

  1. Open Dashboard
  2. Go to Content tab
  3. Find the content you want to edit
  4. Update the text in the input fields
  5. Click "Update"
  6. Check Live Preview to see changes

You can edit:

  • Hero title and subtitle
  • Hero description
  • About section text
  • Vision statement
  • Call-to-action button text

Tutorial 4: Control Page Visibility

  1. Open Dashboard
  2. Go to Pages tab
  3. You'll see all available pages listed
  4. Check/uncheck the checkbox to show/hide pages
  5. Hidden pages won't appear in navigation menu
  6. Changes save automatically
üìå Note: Hidden pages are not deleted, just not visible. You can show them again anytime!

Tutorial 5: Adjust Animations

  1. Open Dashboard
  2. Go to Animations tab
  3. Toggle "Enable All Animations" on/off
  4. Adjust Animation Speed (Slow/Normal/Fast)
  5. Toggle individual animation types on/off
  6. Click preview box to see animation in action
  7. Changes apply instantly to website
üí° For slow internet: Consider disabling animations or setting to "Slow" speed for better performance.

Tutorial 6: Change Language Translations

  1. Open Dashboard
  2. Go to Languages tab
  3. Choose English Translations or Arabic Translations
  4. Click on any field to edit text
  5. Make your changes
  6. Click "Save All"
  7. Website updates with new translations
üåç RTL Support: Arabic text is automatically displayed right-to-left. Just edit and save!

üíæ Data & Storage

How Your Data is Stored

All your changes are saved to browser's LocalStorage:

  • ‚úÖ No server needed
  • ‚úÖ No backend required
  • ‚úÖ Data stays on your computer
  • ‚úÖ Instant save (no manual saving)
  • ‚úÖ Persists across browser sessions
‚ö†Ô∏è Warning: If you clear browser data/cache, your changes will be lost. Export your data regularly!

Backup Your Settings

  1. Open browser DevTools (F12)
  2. Go to Application tab
  3. Find LocalStorage
  4. Copy all key-value pairs
  5. Save to a text file as backup

‚ùì FAQ

Q: Can I undo changes?

A: Yes! Changes are saved automatically. To undo, just change values back to previous settings. Or click "Clear All Data" to reset to defaults.

Q: Will changes affect other users?

A: No. Each visitor's browser stores data locally. Changes you make affect only your copy. To affect all visitors, deploy changes to your server.

Q: Can I add new pages?

A: Yes! Edit index.html to add new sections and update main.js to include them in navigation. Current dashboard toggles existing pages.

Q: Is my contact form data secure?

A: Yes. Data is stored locally in your browser only. Never sent to any server. No one else can see it unless they have access to your computer.

Q: Can I export my settings?

A: Yes! Use browser DevTools to export LocalStorage data. Save as JSON file for backup or sharing with team members.
